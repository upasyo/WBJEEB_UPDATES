# WBJEE Monitoring System

An AI-powered monitoring system for the WBJEE website.

## Setup
1. Clone this repository.
2. Set up GitHub Secrets:
   - `GEMINI_API_KEY`
   - `NTFY_TOPIC`
3. Install requirements: `pip install -r python-monitor/requirements.txt`
4. Run: `python python-monitor/monitor.py`
